Examples for AlphaPlantImpute2
==============================

A number of examples (with data) illustrating the use of AlphaPlantImpute2. Each directory contains data and a script (example.sh) that contains the commands. 
Examples are:

pop     - population imputation (PLINK .ped format)
cross   - biparental cross imputation using (PLINK .ped format)
library - updating a haplotype library with new genotypes (PLINK .ped format)
geno    - population imputation (AlphaImpute .genotypes format)
